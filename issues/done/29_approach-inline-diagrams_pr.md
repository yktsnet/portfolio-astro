## 変更内容

現在 Approach 02（放任より制御）だけに「設計→実装→検証」の3カードフロー図があった。
01（機能より定着）と 03（多機能より安定性）にも同レベルのインライン図を追加し、3つの原則すべてで視覚的に揃えた。

### 追加したフロー図

- **01 機能より定着**（3カード）: 既存の運用 → 裏に差し込む → 手順は変わらない
- **02 放任より制御**（3カード、既存）: 設計 → 実装 → 検証
- **03 多機能より安定性**（2カード対比）: React UI 版 → JSONL + systemd

### 実装の変更点

- `Role` 型を `FlowStep` にリネーム
- `roles` 配列を `flow02` にリネーム
- `flow01`・`flow03` を新規追加
- `Principle` 型に `flow?: FlowStep[]` を追加
- `i === 1` の条件分岐を `p.flow &&` チェックに置き換え（各原則が自前の flow を持つ構造に変更）
