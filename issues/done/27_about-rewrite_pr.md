## 変更内容

About の文章を「思想の表明」から「事実で語る実務家の紹介」へ作り直す。
ポートフォリオの目的（採用側に固有の強みと現在地を最短で伝える）に対し、現状の3点のズレを是正する。

**1. 見出し**
一般論スローガン「曖昧なビジネス課題を、設計・実装・本番運用まで一人で形にする。」を廃し、経歴の事実で固有性を立てる。
→「学習塾を12年経営し、2025年にエンジニアへ転身した。技術が「導入される側」を知っているエンジニア。」

**2. リード文**
段落順を「経営の景色 → 現在」から「現在の二枚看板 → 経営という裏付け」へ反転。
第1段落でテックリード／外部CTO を事実として格上げし、第2段落は経営経験を強みの根拠として圧縮。「一気通貫」を使わず「単独で落とし込む」と言い換え。

**3. 年表**
地理軸（Kyoto / Osaka）を役割軸（Management / Engineering）へ変更。
型・描画ロジックは変えず、配列の中身とラベルのみ差し替え。合同会社の記述を「設立→清算」と正確化し、Engineering グループに外部CTO・テックリードを明記。
