# ykts.net

Personal portfolio and technical blog. Built with Astro, deployed on Cloudflare Pages.

→ **[ykts.net](https://ykts.net/)**

## Stack

- [Astro](https://astro.build/) + Cloudflare Pages / Workers
- [Tailwind CSS](https://tailwindcss.com/)
- [Hono](https://hono.dev/) for API routes
- [Pagefind](https://pagefind.app/) for full-text search

## Development

```bash
npm install
npm run dev
```
