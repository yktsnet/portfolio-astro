## カラーパレット

ダークモードをベースに設計しています。

| Role    | Light           | Dark            |
| ------- | --------------- | --------------- |
| Base    | `zinc-50`       | `#1b1e28`       |
| Text    | `zinc-900`      | `#e4f0fb`       |
| Accent  | `#d0679d`       | `#5de4c7`       |
| Muted   | `zinc-500`      | `#a6accd`       |

## タイポグラフィ

本文には `Helvetica Neue` / `Hiragino Sans` のシステムフォントスタック、コードには `Fira Code` を使用しています。

> デザインはコードで定義される。コードはデザインそのものである。
